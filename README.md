# MyCalendar
## これは何

あったらいいなと思うカレンダー

## 背景

・毎日行うが時間が決まっていない予定は記録、管理が大変。<br>
・特に睡眠は、毎日だいたいしか決まっていない。

## ターゲット

・自分の睡眠のリズムを知りたい人
・睡眠も管理したい人

## 実装したい機能
### 管理したいもの

・自分の予定(仕事用プライベート用)
・他の人の予定
・睡眠時間

### 最低要件

・ログイン、ログアウト
・月でのカレンダー可視化
・日毎のカレンダー可視化
・予定の追加
・予定の削除
・予定の変更

### やりたいこと

・推奨睡眠時間のレコメンド
・その日の睡眠時間(自動で取りたい)
・その日のパフォーマンスの振り返り
